# Testimonials Grid Section

A responsive testimonials grid layout built with HTML and CSS using Flexbox and CSS Grid.

## Project Overview
This project is a static web layout that displays user testimonials in a visually structured grid. The design adapts to different screen sizes using modern CSS layout techniques and focuses on clean typography, spacing, and responsiveness.

The project was completed as part of a web development course and is based on a Frontend Mentor design challenge.

## Technologies Used
- HTML5 (semantic markup)
- CSS3
  - CSS Grid
  - Flexbox
  - CSS variables
  - Media queries
- Google Fonts (Barlow Semi Condensed)

## Features
- Responsive grid layout that adapts across screen sizes
- Combination of Flexbox and CSS Grid for layout control
- Custom color palette using CSS variables
- Profile cards with images and typography hierarchy
- Mobile-first design approach

## What I Learned
- Structuring responsive layouts with CSS Grid
- Combining Grid and Flexbox effectively
- Using CSS variables for maintainable styling
- Implementing media queries for multi-device layouts
- Translating a design mockup into clean, semantic code

## How to View the Project
1. Clone the repository
2. Open `index.html` in a web browser  
   **or**
3. Use a live server extension (such as VS Code Live Server)
